Quick Start

三步开始

  1. 右上角点 `连接设置`，填写接口地址和 Token。
  2. 配置页里先点 `测试连接`，再点 `测试图片模型`，确认这组配置真的能用。
  3. 回到 `文生图` 或 `图生图 / 局部改图` 导航开始生成，右上角助手只负责整理提示词。
Mode Guide

怎么选模式

  1. `文生图`：从零开始，只有提示词，没有参考图。
  2. `图生图 / 局部改图`：你已经有参考图，想延续它的风格、构图或局部修改。
  3. 历史区任何图片都可以一键放进图生图当参考。
Best Practice

推荐用法

  1. 先定左侧尺寸、质量和模式，再去右侧让助手写提示词，这样不会出现左边选 1080p、右边还写 4K 的冲突。
  2. 如果只是试创意，先用 1K / standard；确定方向后再切到更高分辨率成片。
  3. 历史区分页浏览，选中满意结果后可直接继续导出、删除或做图生图参考。

Compatibility

使用前提

  • 接口必须允许浏览器直连，否则会被 CORS 拦住。
  • 推荐 Chromium：可以手选文件夹后批量写入导出。
  • 首次打开无依赖：直接双击 `index.html` 就能用。
  • 503 测试连通：你给的那类后台检测逻辑不适用于图片模型，真实调用看本工具结果即可。

First Run

第一次使用向导

Step 1

先配接口

点开连接设置，填好地址和 Token。你只要知道这两个，其他默认就够用。

Step 2

先测能不能通

在配置窗里先点文本连接测试，再点图片模型测试，确认不是地址或权限问题。

Step 3

再选模式生成

文生图适合从零开始；图生图适合拿参考图延续风格。右侧助手会自动跟随左侧参数。
